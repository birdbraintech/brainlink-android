The Brainlink Android software package includes the following sub-directories:

Library Source - The source code to BrainLinkAndroidLib.jar
Frameworks - Two projects that contain skeleton apps to get you started
Examples - source of five examples to view or modify
APKs - The compiled app files for the apps currently on the Android Market

Full instructions for writing apps for Brainlink are available at:
http://www.brainlinksystem.com/android-programming
